# PowerOfTwoApp
Приложение для проверки числа на степень двойки — сравнение двух алгоритмов (метод деления и битовые операции) с измерением производительности и визуализацией результатов

## Функционал
### Проверка числа:
- Метод последовательного деления
- Метод битовых операций
- Подсчёт количества выполненных операций
- Замер времени выполнения с усреднением по 100 итерациям
- Отображение степени числа (например: «8 = 2³»)

### Экспериментальное исследование:
- Построение графиков зависимости времени от размера данных
- Тестирование на 10 степенях двойки: 2¹, 2⁵, 2¹⁰, ..., 2³⁰
- Визуальное сравнение производительности алгоритмов

## Как запустить
### Через Visual Studio
1. Откройте файл `LibraryAPP.sln`
2. Убедитесь, что стартовым проектом установлен WinForms
3. Нажмите `Ctrl + F5` (`Ctrl + Fn + F5`) для запуска без отладки
4. Приложение откроется с двумя вкладками: «Новая книга» и «Магазин»

## Особенности программы
1. Точный замер времени через Stopwatch
2. User-friendly интерфейс
3. Проверка диапазона: от 1 до 2³⁰
4. Обработка некорректного ввода

## Требования
1. .NET 10.0 или новее
2. Visual Studio 2019+ (или любая среда с поддержкой C# и WinForms)
